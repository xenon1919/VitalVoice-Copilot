System Architecture Overview

Agent Roles

Orchestrator: Selects optimal response strategy (direct, retrieval, or tool-driven).
Planner: Interprets intent and defines execution steps.
RAG Agent: Retrieves relevant domain context from vector search.
Tool Agent: Executes structured clinical calculators and checks.
Memory Agent: Persists and retrieves semantically related session history.

Processing Pipeline

Input ingestion (text or voice) -> intent planning -> route selection -> contextual/tool execution -> safety policy checks -> structured response and optional speech output.

Technology Stack

LLM: Google Gemini
Embeddings: MiniLM-L6-v2 (384 dimensions)
Vector Database: FAISS
Speech-to-Text: faster-whisper
Text-to-Speech: Google TTS
Backend Runtime: FastAPI on Python 3.11

Safety and Governance

Urgent symptom patterns are flagged for immediate escalation guidance.
Responses are informational and avoid diagnostic claims.
Context memory supports continuity while preserving conversation-level relevance.
